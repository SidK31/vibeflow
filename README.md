# VibeFlow

VibeFlow is a clean, interactive frontend project focused on UI state management and user interaction using vanilla JavaScript.

This project demonstrates how small UI changes can create meaningful user experiences without using any frameworks.

## Features

- Interactive mood selection with dynamic UI updates
- Active state handling for buttons
- Clean dark-themed interface
- User greeting based on input
- Simple, readable, and maintainable code

The goal of this project is to practice:

- User interaction handling
